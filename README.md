# 🤖 WhatsApp Mass Message Bot

Envie mensagens em massa no WhatsApp automaticamente. Perfeito para irritar amigos!

⚠️ **Aviso:** Use com responsabilidade - spam pode fazer o WhatsApp bloquear sua conta!

---

## ⚡ Como usar

1. **Instale** `pip install pyautogui`
2. **Execute** `python bot.py`
3. **Abra** web.whatsapp.com e escaneie o QR Code
4. **Clique** na caixa de mensagem quando o bot pedir
5. **Escolha** mensagem, quantidade e veja a mágica! 🎯

---

## ✨ Funcionalidades

| Funcionalidade | Descrição |
|----------------|-----------|
| 🎲 Personalizada | Mensagem diferente a cada envio |
| ⚡ Controle de velocidade | Ajuste o intervalo entre mensagens |
| 📊 Progresso | Acompanhe quantas já foram enviadas |
| 🛑 Interrupção | Pressione CTRL+C para parar |
